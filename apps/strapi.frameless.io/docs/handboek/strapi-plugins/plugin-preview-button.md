# Preview Button

## Wat doet deze plugin?

Met de Preview Button kan de redactie een voorbeeld bekijken van hoe content eruitziet in het KISS dashboard.

De plugin is op dit moment gekoppeld aan de Overige Objecten API en wordt gebruikt voor:

- **Producten** — in het PDC dashboard
- **VAC** — in het PDC dashboard

---

## Hoe gebruik je de Preview Button?

### 1. Open een product

Content Manager -> **Producten** -> item aanmaken of open het product dat je wilt bekijken

### 2. Klik op Open preview

Rechts in de zijbalk zie je de Item block en klik op **KISS-voorbeeld**.
In de rechterzijbalk vind je het Item block met knopjes. Klik hier op **KISS-voorbeeld** knop om de KISS preview te bekijken.

![Bekijk voorbeeld](/img/strapi-plugins/strapi-dashboard-product-KISS-preview-knop.png)

De preview opent als een nieuw tabblad binnen het KISS dashboard.

### 3. Bekijk de preview

Bovenin de preview zie je:

- Het **Gepubliceerd** label, waarmee wordt aangegeven dat de content gepubliceerd is.
- De **Kopiëren** knop, waarmee je de volledige content kunt kopiëren.

Aan de linkerkant staat een zijbalk met de kop Categorieën. Onder deze kop zie je verschillende categorieknoppen, zoals Inleiding, Aanvraag en Kosten.

Door op een categorie te klikken, wordt de content gefilterd op die categorie. Klik je bijvoorbeeld op Aanvraag, dan wordt alleen de content uit de categorie Aanvraag weergegeven. Klik je op Kosten, dan zie je uitsluitend de informatie die onder Kosten valt. Op deze manier kun je snel navigeren naar de informatie die je zoekt.

![Bekijk voorbeeld](/img/strapi-plugins/strapi-dashboard-KISS-preview.png)
